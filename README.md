# RN模板项目

#  RN-项目结构简介
1. RN版本：0.61.2

## 安装依赖
`yarn install`

## 启动命令
`yarn android`

## 入口文件
* index.js

## 项目目录
* view （详情见目录下方 `README.md` 文件）

| 目录名 | 别名 |
| --- | --- |
| ./view/resource/images | @images |
| ./view/pages | @pages |
| ./view/config/style | @style |
| ./view/config/i18n | @i18n |
| ./view/config/fetch | @api |
| ./view/config/methods | @Com |
| ./view/components | @components |
